# 交易所批量提币

### 功能概述

在加密货币生态中，交易所的钱包地址通常被项目方列入白名单，这使得从交易所批量提币到多个钱包成为空投策略的关键环节。传统提币流程需要反复输入验证码，既耗时又繁琐。CPBOX推出的【[交易所批量提币](https://www.cpbox.io/cn/exchange/withdraw?_s=docs)】功能专为解决这一问题而设计，目前支持Binance（币安）、OKEx、Bitget、Gate.io和Bybit五大交易所，让您能够高效地批量提币到链上地址。如果您有其它交易所需求，也可以联系客服进行定制。

### 支持的交易所

CPBOX目前支持以下五大交易所的批量提币功能：

1. **[Binance（币安）](https://binance.com/)**
2. **[OKEx](https://okx.com/)**
3. **[Bitget](https://www.bitget.com/)**
4. **[Gate.io](https://www.gate.io/)**
5. **[Bybit](https://www.bybit.com/)**

### 通用操作流程

无论您使用哪个交易所，批量提币的基本流程都包括两个主要部分：

1. 在相应交易所创建API密钥并设置提币权限
2. 在CPBOX平台配置批量提币参数并执行操作

### 以OKEx 为例

#### 第一部分：创建API密钥

1. 登录OKEx官网
2. 进入"我的账户" > "API管理"
3. 点击"新建API"
4. 设置API名称和权限（确保包含提币权限）
5. 添加IP白名单（从CPBOX获取）

<figure><img src="../../.gitbook/assets/cex-withdraw.jpg" alt=""><figcaption><p>你的IP</p></figcaption></figure>

1. 完成验证并保存API Key与Secret Key

#### 第二部分：CPBOX批量提币设置

1. 登录CPBOX，选择"[交易所批量提币](https://www.cpbox.io/cn/exchange/withdraw?_s=docs)"
2. 选择"OKEx"交易所
3. 输入API密钥信息
4. 获取并选择资产和网络
5. 导入接收地址
6. 设置提币参数
7. 确认并提交

### 批量提币通用技巧

1. **API密钥安全**：创建专用API，仅开启必要的提币权限，使用后及时删除
2. **分批操作**：对于大量提币，建议分批进行以降低风险
3. **地址核对**：提币前仔细核对地址格式是否正确
4. **网络选择**：确保选择正确的区块链网络，避免资产丢失
5. **时间间隔**：根据交易所限制合理设置提币间隔
6. **提币限额**：了解各交易所的每日提币限额，合理安排提币计划

## 定制IP功能

主流交易所为保障账户与资金安全，通常会对 **API 提币 / 提现** 设置 **IP 白名单**：只有来自指定 IP 的请求才会被接受。

CPBOX 的交易所批量提币通过定制IP访问交易所，您会在页面上看到 **当前提币出口 IP**（用于在交易所后台配置白名单）。默认情况下，出口 IP 与**机房** 绑定；不同用户对 **合规、延迟、交易所策略** 的需求不同，单一区域未必能满足所有人，因此会有定制IP的需求。

|             |                                          |
| :---------- | :--------------------------------------- |
| 场景          | 说明                                       |
| 交易所策略与区域限制  | 部分交易所对特定地区 API 或提现策略不同，需要将出口 IP 落在可接受区域。 |
| 合规与风控要求     | 团队或机构对数据出境、访问来源地有明确要求，需与业务所在地或约定区域一致。    |
| 网络与稳定性      | 希望出口更靠近目标交易所或常用链路，以降低异常或延迟（需以实际支持为准）。    |
| 多账户 / 多策略隔离 | 不同子业务希望使用不同IP出口，便于在交易所分 IP 管理            |

* 定制入口 **仅对具备交易所批量提币 VIP 权限的用户展示**（账号需开通对应 VIP 能力）。
* 非 VIP 用户界面中 **不显示**「定制 IP」按钮；如需使用请先了解 VIP 开通方式。

### 如何使用定制IP

1. **进入** 站内 **交易所批量提币** 页面。
2. 在交易所选择区域 **右上角**（移动端可能在标题行下方），点击 **「定制 IP」** 按钮，打开 **「定制提币 IP」** 弹窗。
3. 按表单填写并提交：
4. 点击「提交」按钮后，系统将即时依托智能算法与高效底层系统进行动态IP地址分配，绝大多数定制需求将由系统自动完成 IP 配置与适配，全程无需人工干预，高效省心；
5. 若您的需求涉及特殊复杂场景（如高安全级别的专属 IP 池、跨多区域的精细化部署等特殊定制场景），我们的专属技术顾问将主动通过您预留的联系方式与您对接，精准确认细节并提供定制化配置支持；
